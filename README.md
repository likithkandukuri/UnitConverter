# Unit Converter

A simple yet comprehensive Python unit converter built to practice coding, logic development, and input handling.  
This version supports multiple measurement categories with clear output and an interactive console menu.

## Features

- Converts **weight, distance, temperature, volume, speed, area, time, pressure, energy, and data**
- Offers **accurate conversion formulas** with clean, readable results
- Provides **input validation** and prevents invalid or incorrect entries
- Allows users to perform **continuous conversions** without restarting the program
- Simple and **easy-to-use console interface**

## Requirements

- **Python 3.8+**
